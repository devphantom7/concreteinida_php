Key Responsibilities

Concretech India is rapidly expanding, and we're seeking talented and motivated Software Engineers to join our Web Development team. As a Software Engineer at Concretech India, you will play a key role in building innovative, high-performance web applications and platforms that support the construction industry. You will work closely with a diverse team of engineers, product managers, and designers to deliver cutting-edge digital solutions.


Job Description

Develop and maintain modern web applications: Build scalable, efficient, and fast single-page web applications using technologies such as React, Node.js, and PHP.

API Development: Design and develop robust REST APIs that support our applications and integrate with various third-party services.

Collaborate across teams: Work closely with cross-functional teams, including product managers, designers, and other engineers to deliver the best products for our clients.

Write clean, efficient, and maintainable code: PrFollow best practices for coding standards, testing, and version control, ensuring high-quality software.

Contribute to the architectural decisions: Provide input on architecture and design patterns, helping shape the direction of the projects.

Optimize performance: Continuously monitor and optimize applications for performance, speed, and scalability.

Ensure responsive design: Implement responsive web designs using tools like SCSS, ensuring seamless experiences across devices.


Required Skills and Qualifications:

We are looking for someone with at least 2 years of experience in web development and a solid portfolio of web applications. You should be skilled in JavaScript, including ES6+ and modern frameworks like React. Experience with backend technologies like Node.js or PHP is also needed. You should understand how to design and work with REST APIs. Familiarity with CSS/SCSS and responsive design is required. You should know how to use Git for version control and be good at solving problems with web applications and APIs. Strong communication and teamwork skills are important, as you will be working with other team members to create great products.
